# 📊 Sales Dashboard Project (Sample Dataset)

## 📌 Overview
This project focuses on analyzing and visualizing sales data using Excel/Google Sheets. The dataset contains information about orders, products, regions, and sales performance. The goal is to transform raw data into meaningful insights through a structured dashboard.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing  
- Analyze sales data to identify trends  
- Create an interactive dashboard  
- Generate business insights for decision-making  

---

## 📂 Dataset Description
The dataset includes the following columns:
- Order_ID → Unique order number  
- Date → Order date  
- Product → Product name  
- Region → Sales region  
- Sales → Revenue generated  
- Quantity → Number of items sold  

---

## 🛠️ Tools Used
- Excel / Google Sheets  
- Pivot Tables  
- Charts (Bar, Pie, Line)  
- Data Cleaning Techniques  
- Slicers (Filters)  

---

## 📊 Key Features of Dashboard
- 💰 Total Sales Overview  
- 🌍 Region-wise Sales Analysis  
- 📦 Product-wise Performance  
- 📈 Monthly Sales Trends  
- 🎛️ Interactive Filters (Slicers)  

---

## 🔍 Data Cleaning Steps
- Removed duplicate records  
- Handled missing values  
- Standardized column formats  
- Sorted and filtered data  

---

## 📈 Insights Gained
- Identified top-performing regions  
- Found highest-selling products  
- Observed monthly sales growth trends  
- Compared performance across different categories
